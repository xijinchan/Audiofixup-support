# Audiofixup-support

This is the official issue tracker for Audiofixup Chrome browser extension.

## How to Report an Issue

1. Check the [existing issues](https://github.com/xijinchan/Audiofixup-support/issues) first (someone may have already reported it).
2. Create a [new issue](https://github.com/xijinchan/Audiofixup-support/issues/new/choose).
3. Use the appropriate template (Bug Report / Blank).
